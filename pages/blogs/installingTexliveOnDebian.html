Understanding the available module to test your Python code

In order to run GUI-based installation, install 'perl-tk' package

Debian/Ubuntu 

                        sudo apt install perl-tk
Red Hat/CentOS 

                        yum install perl-tk

Mount .iso file in the appropriate path, e.g. /mnt, you will get error unless you are root user


                    sudo mount /path/to/.iso /mnt

Change directory to the /mnt folder and run the following command


                        cd /mnt
                        ./install-tl --gui=perl-tk

Follow the installation instructions as your requirements
