Installing Docker CE

Installing Docker CE on Debian is straightforward, although Debian does not provide any prepared packages in its repository.

1. Setup the repository

Unlikely, at the time of writing this blog, there is no active repository on Debian repoisotry list. Hence, we should use the one which Docker offers. The task of adding the repository starts by updating the apt.

Then we should install several prerequisite packages, which will be needed during the installation. 

$ apt-get install apt-transport-https ca-certificates curl gnupg2 software-properties-common

Now setting up the repository is somehow complete. However, there are a couple of more steps to install Docker CE.

2. Adding official's GPG key To prevent further authentication error, it is better to add official GPG key. 

$ curl -fsSL https://download.docker.com/linux/debian/gpg | sudo apt-key add -

Adding stable repository to repository list; use the following command to add the Docker hosting package repository. 

$ sudo add-apt-repository \
           "deb [arch=amd64] https://download.docker.com/linux/debian \
           $(lsb_release -cs) \
           stable"

3. Install Docker CE

To make changes to be effected on system update the apt package manager. 

$ sudo apt-get update

Lastly, install Docker CE by running the following command on the terminal:

$ sudo apt-get install docker-ce

The installation process is complete. Now, unless you do not get any error, follow the instruction How to use Docker CE. Otherwise, please Google the exact error.
